# Painel de Suplentes — Dra. Fernanda Sarelli

Painel de gestão de suplentes e campanha política para Aparecida de Goiânia.

## Tech Stack

- React 18 + TypeScript
- Vite + Tailwind CSS
- shadcn/ui (Radix UI)
- Supabase (banco de dados e autenticação)
- TanStack Query
- React Router DOM

## Desenvolvimento

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Testes

```bash
npm run test          # unit tests (vitest)
npx playwright test   # e2e tests (playwright)
```
